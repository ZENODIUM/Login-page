# Login-page

By S.Vishnu

About:
this is a simple login web applicationusing flask which has register and forget password functions
It uses sqlite to store the database of users an their respective login credentails 
Flask-Login provides user session management for Flask. It handles the common tasks of logging in, logging out, and remembering your users' sessions over extended periods of time.

Usage:
At first you will be reirected to the home page where yuo needed to register and fill in details
for login, you need to redirect back to homepage by navigation link and login to the website
In case if you have forgotten the password, you can retrieve it by using additional key in forgot password page

Languages and frameworks used:-
Python (Flask)
Fornt end - Html, CSS
Bootstrap and jquery
